04

密码安全设置

设置强密码并启用安全功能：

强密码要求 
密码长度: 至少12位字符
必须包含:
  ✓ 大写字母 (A-Z)
  ✓ 小写字母 (a-z)
  ✓ 数字 (0-9)
  ✓ 特殊字符 (!@#$%^&*)

避免使用:
  ✗ 个人信息（生日、姓名）
  ✗ 常见词汇（password、123456）
  ✗ 连续字符（abcdef、123456）
✓
启用双因素认证

在账户设置中开启 2FA，增加安全层级

✓
定期更换密码

建议每3个月更换一次密码
