01

了解密码类型

首先了解 Clodesen 系统中使用的密码类型：

✓
访问密码

用于登录系统的主要凭证，需要妥善保管

✓
临时密码

系统自动生成的一次性密码，用于首次登录或密码重置

✓
API 密钥

用于程序化访问的密钥，具有特定权限范围

💡

提示：不同类型的密码有不同的安全级别和使用场景，请根据实际需求选择合适的密码类型。

02

获取访问密码

通过以下方式获取访问密码：

1

访问 Clodesen 官方网站或管理后台

2

在登录页面点击 [忘记密码] 或 [获取密码]

3

输入注册时使用的邮箱地址

4

检查邮箱，获取系统发送的密码信息

密码邮件示例 
主题: Clodesen 访问密码

您的访问密码已重置：
用户名: your_username
临时密码: abc123XYZ
有效期: 24小时

请登录后立即修改密码。
03

首次登录

使用获取的密码进行首次登录：

1

打开 Clodesen 登录页面

2

输入用户名和临时密码

3

系统会提示你修改密码

4

设置新的强密码并确认

ℹ️

安全提示：首次登录后请立即修改密码，临时密码将在24小时后失效。

04

密码安全设置

设置强密码并启用安全功能：

强密码要求 
密码长度: 至少12位字符
必须包含:
  ✓ 大写字母 (A-Z)
  ✓ 小写字母 (a-z)
  ✓ 数字 (0-9)
  ✓ 特殊字符 (!@#$%^&*)

避免使用:
  ✗ 个人信息（生日、姓名）
  ✗ 常见词汇（password、123456）
  ✗ 连续字符（abcdef、123456）
✓
启用双因素认证

在账户设置中开启 2FA，增加安全层级

✓
定期更换密码

建议每3个月更换一次密码

05

密码管理与恢复

妥善管理密码并了解恢复流程：

密码管理建议 
推荐密码管理器:
  • 1Password
  • LastPass
  • Bitwarden

密码存储原则:
  ✓ 使用加密的密码管理器
  ✓ 不要在多个网站使用相同密码
  ✓ 不要将密码写在不安全的地方
  ✓ 定期备份重要凭证
